====================
1. My list:
====================

Import module:
    >>> MyList = __import__('1-my_list').MyList

Class:
=======.
Write a class that inherits from list.

Operations:
=========
l_a and l_b are lists:

normal append:
    >>> my_list = MyList()
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> print(my_list)
    [4, 2]

normal unvalid element:
    >>> my_list = MyList([3, 4, 6])
    >>> my_list.append("hola")
    >>> print(my_list)
    [3, 4, 6, 'hola']

unvalid list method:
    >>> my_list = MyList([3, 4, 6])
    >>> my_list.call()
    Traceback (most recent call last):
    AttributeError: 'MyList' object has no attribute 'call'

clear method:
    >>> my_list = MyList([3, 4, 6])
    >>> my_list.clear()
    >>> print(my_list)
    []
